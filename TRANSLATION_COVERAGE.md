### Usage in Components
```typescript
const { t } = useSettings();

// Simple translation
<h1>{t('heroTitle')}</h1>

// Dynamic translation
<h2>{t(activeService)}</h2>
<p>{t(`${activeService}Desc`)}</p>

// Category translation
{translateCategoryLabel(category, language)}

// Product keyword translation
{translateProductLabel(productName, language)}
```

### Persistence
- Language selection saved to `localStorage`
- Persists across page refreshes
- Persists across sessions
- No re-selection needed
